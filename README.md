# ejercicio parcial 1
Mi primer proyecto de React con Expo GO


1. Crear proyecto: **npx create-expo-app parcial1 --template**
2. Seleccionar **Blank / En blanco**
3. Escribir el nombre de tu aplicación
4. Crear una vista de navegación con Tabs o Pestañas donde los botones tengan el mismo width y estén acomodados en una fila al fondo de la página.
5. Crear una navegación donde al hacer clic en cada tab muestre el nombre del componente. Enviar este titulo de componente a través del parámetro en la ruta.
